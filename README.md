# Crypto Track App

## React SPA that track values of various cryptocurrencies.

### Installation:
if you are using yarn, run:
````
yarn install
yarn serve
````
or, for npm:
````
npm install
npm run serve
````
inside the cloned repository. Then navigate to:
````
http://localhost:4200
````
To build production version of the app, run:
````
yarn build
````
or
````
npm run build
````
